Op Pad Met Code

Op Pad Met Code neemt je mee in de wereld van P5.js! Met dit online programma kun je door middel van code creëren wat je maar wilt. Tijdens onze workshop leer je in maximaal 3 uurtjes wat code en P5.js inhoudt, wat je hiermee kunt maken en hoe. Door middel van wat theorie, oefen voorbeelden en een creatieve eindopdracht krijg je P5.js zo onder de knie!

Beginnersniveau

3 uur

30 personen

11-14 jaar

Waarom P5.js?

P5.js is een Javascript bibliotheek die toegankelijk is voor iedereen. Het is speciaal ontworpen zodat beginners gemakkelijk kunnen experimenteren met tekenen en animeren door middel van code. De perfecte tool voor creatievelingen zonder codeer ervaring!
