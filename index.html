Waarom MiniBytes?

Wij als MiniBytes zetten ons in voor de nieuwe generatie van digitale creatievelingen. Wij willen, door middel van onze workshops, kinderen van alle leeftijden bewust maken van het belang van code en hen op een speelse en creatieve manier kennis laten maken met het vak. Coderen hoeft niet lastig te zijn, als je het leert in kleine hapjes!

Code om ons heen

Wij leven in een wereld, waarin vrijwel alles gerund wordt door code. Van je wekker, tot aan de verkeerslichten op straat. En wat dacht je van het digibord in je klaslokaal? Code is op zijn simpelst gezegd, een reeks instructies die een computer vertelt wat hij moet doen. Het is overal, je kunt er niet aan ontsnappen. En waarom zou je dat ook willen?


Wist je dat het leren van code waardevolle vaardigheden met zich meebrengt?¬† Het is meer dan alleen cijfertjes en lettertjes. Het bevordert het probleem-oplossingsvermogen, rekenskills, de creativiteit en nog veel meer. Jouw fantasie is de limiet!

Onze workshops

Steeds meer scholen en organisaties beginnen het belang van code in te zien. Daarom zijn wij als MiniBytes op steeds meer plekken te vinden! Hieronder vindt je de workshops die wij aanbieden, net als leuke beelden en resultaten van voorgaande workshops.

Lego Robots

Maak met behulp van je laptop een eigen Lego-robot, die je zelf kunt aansturen en obstakels kan laten overbruggen!

Op Pad met Code

Op Pad Met Code neemt je mee in de wereld van P5.js! leer in 3 uurtjes hoe je creatief te werk gaat met code.

Arcade Games

Maak je eigen arcadegame! Met deze workshop gaan we met P5.js aan de slag om een eigen spel te maken.
